# 🧠 Quiz Application (Java Swing)

A GUI-based **Quiz Application** built using **Java Swing**, inspired by the popular quiz format. This application provides an interactive multiple-choice quiz experience with features like a countdown timer, lifelines, and reward checkpoints.

---

## 🚀 Features

* 🎯 Multiple-choice quiz (MCQs)
* ⏱️ 40-second countdown timer for each question
* 💰 Reward system with safe checkpoints
* 🛟 Lifelines:

  * 50-50 (removes two wrong options)
  * Audience Poll (suggests correct answer)
  * Skip Question
* ❌ Quit option anytime with safe winnings
* 🎨 Simple and clean GUI using Java Swing

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* AWT (Event Handling)

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/quiz-application.git
   ```

2. Navigate to the project folder:

   ```
   cd quiz-application
   ```

3. Compile the Java file:

   ```
   javac project/KBCAdvanced.java
   ```

4. Run the application:

   ```
   java project.KBCAdvanced
   ```

---

## 🎮 How It Works

* The quiz consists of **10 questions**
* Each question has **4 options (A, B, C, D)**
* You have **40 seconds** to answer each question
* Correct answers move you forward
* Wrong answer ends the game
* Safe checkpoints:

  * Question 5
  * Question 7
* Lifelines can be used only once each

---

## 🛟 Lifelines Explained

| Lifeline | Description                   |
| -------- | ----------------------------- |
| 50-50    | Removes two incorrect options |
| Audience | Suggests the correct answer   |
| Skip     | Skips the current question    |
